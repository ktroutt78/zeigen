Zeigen — Design System

Tokens distilled from the four canonical surfaces (capture window, floating webcam, tray, post-record review). Restrained, dense, macOS-native. Dark only. One accent (electric blue) for primary actions; recording red is reserved for live-state, never brand. Typography is system-font-stack. Hairline borders over solid; soft shadows over heavy elevation.

Principles

Hairlines, not boxes

Default to 1px rgba(255,255,255,0.06–0.14) separators. Reach for solid borders only on inputs. Never both border + shadow on the same surface.

One accent, one signal

Electric blue is the only chromatic action color. Recording red is reserved for live state. Success green only confirms (saved, connected). Anything else stays neutral gray.

Density first, then breathe

Rows are 22–36px. Padding is tight (9–14px). Use whitespace deliberately — a single 18px gap is more meaningful than three 12px gaps.

System-tonal type

System font stack only. Tracking goes negative as size grows (-0.005 → -0.015em). Monospace only for time, size, file paths, and keyboard hints — never body.

Glyphs over labels

SF-Symbols-style 1.25–1.5px stroke icons. Pair an icon with a label only when the action is destructive or unfamiliar. Tertiary fg by default; primary fg on hover.

Translucency for tray, not chrome

Backdrop-blur belongs on the menu-bar tray and on overlays inside video/camera surfaces. Main app windows are opaque — no glassmorphism on the chrome itself.
